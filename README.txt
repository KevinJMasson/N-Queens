Program problem formulation:

X = {Q1, Q2, . . . Qn} in which N is the number of queens you wish to add.
D = {n*n} in which a queen can be placed in any spot on the N*N board
C = {A queen cannot be placed in the same column as another queen, A queen cannot be placed in the same
	row as another queen, A queen cannot be placed diagonally from another}
